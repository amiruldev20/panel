2. PERBEDAAN SERVER FREE DAN VIP
  #SERVER VIP
  - JET BACKUP, AUTO BACKUP PERHARI. DIPASTIKAN DATA AMAN

  - SPEED LEBIH LANCAR
  
  #SERVER FREE
  - TIDAK ADA BACKUP (ANDA HARUS BACKUP SCRIPT SENDIRI)
  - SERVER KAPASITAS LEBIH KECIL
  - SPEED DIBAWAH SERVER VIP
  - INGIN AUTO BACKUP? DONASI
